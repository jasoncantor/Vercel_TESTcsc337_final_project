Help Page

Welcome to the Golf Course Management System! This page will guide you through the various features of the system.

Login

To log in, enter your username and password on the login screen. If you don't have an account, you can create one by entering a new username and password.

Dashboard

Once logged in, you will be taken to the dashboard. The dashboard contains several sections:

  • Job Board: View and manage tasks assigned to you or others.
  • Special Task Tracker: Track special tasks and their progress.
  • Budget Tracker: Manage and track budget expenses.
  • Equipment Tracker: Track and manage equipment inventory.
  • Messaging: Send and receive messages from other users.
  • Help: Access this help page for guidance.

Job Board

The Job Board allows you to view and manage tasks. You can create new tasks, mark tasks as completed, edit tasks, and delete tasks.

Special Task Tracker

The Special Task Tracker allows you to track special tasks. You can add new special tasks and delete existing ones.

Budget Tracker

The Budget Tracker allows you to manage and track budget expenses. You can add new expenses and delete existing ones.

Equipment Tracker

The Equipment Tracker allows you to track and manage equipment inventory. You can add new equipment and delete existing equipment.

Messaging

The Messaging section allows you to send and receive messages from other users. You can view messages sent to you and delete messages.

Logout

To log out, click the "Logout" button on the dashboard. This will return you to the login screen.
